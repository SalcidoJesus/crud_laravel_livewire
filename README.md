# comandos de laravel
### comandos para crear componentes, modelos, etc

este comando es para crear un componente
"Alerta" es el nombre del componente
```bash
php artisan make:component Alerta
```

para mandarlo llamar, es así

```html
<x-alerta/>
```

o así

```html
<x-alerta></x-alerta>
```
